1. What is JSX, and why is it used?
   JSX is an HTML-like syntax that allows developers to write HTML inside JavaScript. With JSX, developers can structure UI code in a way that is cleaner and easier to read.

2. What is the difference between State and Props?
   State is declared inside a component and can be changed. Props are used to send data from a parent component to a child component and cannot be changed by the child.

3. What is the useState hook, and how does it work?
  The useState hook is used to store and update state in a component. It returns a value and a function that is used to update that value.

4. How can you share state between components in React?
  State can be passed from a parent component to a child component using props, and from a child to a parent by lifting up the state.

5. How is event handling done in React?
  Event handling in React is done by passing events using attributes like onClick, onChange, etc., to handle user interactions.
